# Digital Literacy Project

## Overview

This project was completed as part of the Digital Literacy Program. It focuses on improving digital skills, awareness, and responsible use of technology.

---

## Tasks Included

### Task 1 – Digital Awareness

Understood the basics of digital literacy, including safe internet usage, identifying reliable information, and responsible online behavior. This task helped build a strong foundation for all other activities.

### Task 2 – Email Communication

Created professional email samples demonstrating proper format, tone, and structure.

### Task 3 – Social Media Awareness

Developed a checklist for safe and responsible use of social media platforms.

### Task 4 – GitHub & Documentation

Organized project files in a structured repository and learned basic version control using GitHub.

### Task 5 – Cybercrime Awareness

* Case study on UPI/Online Payment Fraud
* Prevention checklist with safety tips
* Awareness of reporting platforms
